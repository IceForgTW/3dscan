//*****************************************Steps*********************************************************************************************************//
This code performs the following:-
1.Generate veritcal & horizontal fringe patterns.
2.Project the vertical & horizontal patterns one by one.
3.Compute the wrapped phase saparately for vertical & horizontal pattern.
4.Computes unwrapped phase for horizontal & vertical pattern.
5.Computes absolute phase & hance camera-projector correspondance.
6.Performs system calibration.
7.Performs triangulation.
